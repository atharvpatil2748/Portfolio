Amazon Clone (Frontend)

A frontend clone of Amazon’s homepage built using HTML and CSS, with a strong focus on layout accuracy, responsiveness, and clean UI structure. This project helped me understand how large-scale websites structure their layouts.

Key Highlights:

  • Implemented modern layouts using Flexbox
  • Focused on pixel-level UI alignment and spacing
  • Built fully responsive design for different screen sizes
  • Deployed live using GitHub Pages

What I Learned: Improved my understanding of HTML semantics, CSS layout systems, and how frontend structure impacts user experience. JavaScript functionality is currently under development.

A research-oriented project offered by the Association of Mechanical Engineering , IIT Kanpur. The project focuses on understanding and optimizing trajectories for robotic systems.

What I’m Learning: This project is strengthening my system-level thinking, bridging theoretical knowledge with real-world robotic applications, and improving my research-oriented problem-solving approach.
